Exam

Q1 : What is JavaScript and what is it used for?

Answer : JavaScript is a high-level, dynamic programming language commonly used for web development. It allows developers to create interactive elements on websites, such as form validation, dynamic content updates, and animations. JavaScript can be used both on the client-side and server-side.

Q2 : What is JavaScript?

Answer : JavaScript is a programming language commonly used in web development to create interactive effects within web browsers. It is versatile and works on both the client-side and server-side.

Q3 : What is the difference between let and var in JavaScript?

Answer : let allows you to declare variables that are limited to the scope of a block statement, while var declares variables globally or locally to an entire function regardless of block scope.

Q4 : What are JavaScript closures?

Answer : A closure is a function that retains access to its outer scope even after the outer function has finished executing. This allows for private variables and encapsulation.

Q5 : What is the purpose of the this keyword in JavaScript?

Answer : The this keyword refers to the object it belongs to. Its value depends on where it is used: in an object method, it refers to the object; alone, it refers to the global object; in a function, it depends on how the function is called.

Q6 : What is an arrow function in JavaScript?

Answer : An arrow function is a shorthand way to write function expressions in JavaScript. It uses the => syntax and does not have its own this context, which is inherited from the parent scope.
